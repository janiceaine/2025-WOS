Intro to Flexbox

Flexbox is a versatile way to align and place elements in one direction.

Flexbox consists of two groups of CSS Properties

  • Flex container properties (parent)
  • Flex item properties (children)

Flex Container Properties

  • justify-content
    • flex-start
    • center
    • flex-end
    • space-around
    • space-between
    • space-evenly
  • align-items
    • flex-start
    • center
    • flex-end
    • baseline

Flex Item Properties

  • align-self
  • flex
